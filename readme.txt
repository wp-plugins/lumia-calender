== Description ==

The Events Calendar is a carefully crafted, extensible plugin that lets you easily share your events

= Plugin Features =

* Admin interface to add, edit and manage Calendar
* Users can add calender events
* Calendar month view with popups

Shortcodes

[event_calender] - For Calendar month view.

Theme Functions

show_event_calender() - Function for Calendar month view.

Support

So that others can share in the answer, please submit your support requests through the WordPress forums for Lumia Calender(http://wordpress.org/support/plugin/wp-lumia-calender).

If you want private or priority support, please [donate](http://weblumia.com/donate) at least $5 USD to cover my time.

== Installation ==

1.     Via WordPress Admin > Plugins > Add New, Upload the wp-lumia-calender.zip file
2.     Alternately, via FTP, upload wp-lumia-calender directory to the /wp-content/plugins/ directory
3.     Activate the 'Lumia Calender' plugin after uploading or through WordPress Admin > Plugins

Usage

1. Add and manage the quotes through the 'Lumia Calender' menu in the WordPress admin area
2. Configure the 'Lumia Calender' to select quotes and display as needed
3. Use the [show_event_calender] or [event_calender] shortcodes to display calendar month view on a page or in a post
